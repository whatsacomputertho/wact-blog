One way to characterize the real numbers \mathbb{R} is to simply take an axiomatic approach. ​ ​That is to say that we will first boil their nature down to a single property or statement from which we can derive the rest of their essential properties. Then we will assume that this property is true without proof, and proceed to build theoretical results atop the property.

To be clear, this is not a sufficient construction of the real numbers since we are simply asserting that some fundamental property is true without justification.  But this is a pragmatic way to eventually build toward a more pure construction.  If we can show that all of the required properties of the real numbers follow from this particular property, then to purify our construction, we just need to show that this particular property follows from some underlying logical foundation.
